# 🧠 FactorLab

A modular research framework for quantitative factor analysis, evaluation, and combination optimization.

---

## 🔍 Overview

**FactorLab** provides an end-to-end pipeline for factor research:
1. **Factor definition** — modular factor construction system
2. **Factor evaluation** — IC, IR, turnover, correlation analysis
3. **Factor combination** — optimization via mean-variance, risk-parity, or regression-based blending
4. **Visualization & reporting**
